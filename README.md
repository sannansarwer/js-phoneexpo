# Phone Expo

A dynamic, responsive web application showcasing smartphones from multiple brands. Users can browse products, search by brand or model, and filter using dropdown menus. Built with HTML, CSS, JavaScript, and Bootstrap 5.

## Demo

You can preview the app locally by opening index.html in your browser.

## Features

- Responsive Layout – Works seamlessly across mobile, tablet, and desktop devices.

- Dynamic Product Cards – Products are displayed using JavaScript and JSON data.

- Live Search – Filter products by brand or model as you type.

- Brand Dropdowns – Quickly filter products by brand using dropdown menus.

## Technologies

HTML5 – Markup and structure.

CSS3 & Bootstrap 5 – Styling and responsive design.

JavaScript (ES6) – Dynamic content rendering and filtering.

JSON – Product data storage and retrieval.

## Usage

Browse Products – Scroll through the product cards to see different phone models.

Search – Type in the search box to filter phones by brand or model.

Filter by Brand – Use the dropdown buttons to display phones of a specific brand.
